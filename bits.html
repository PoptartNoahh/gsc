Topology Stream (angular position, DFS order per face)

The path from root to leaf is the angular position. No explicit angles are stored.

Node TypeBitsLayout
Internal node5[1][C C C C] — flag=1, then 4-bit child mask
Leaf (single point)1[0] — flag=0
Leaf (at max depth)9[0][N N N N N N N N] — flag=0, then 8-bit point count

Payload Stream (distance, DFS order, delta resets per face)

EncodingBitsLayoutRange
Short delta9[0][Z Z Z Z Z Z Z Z] — flag=0, 8-bit zigzag delta±127 quanta (±0.39m at 200m range)
Absolute17[1][D D D D D D D D D D D D D D D D] — flag=1, 16-bit raw distance0 – max_distance

Distance quantization: uint16 linear over [0, max_distance]. Precision = max_distance / 65535. At 200m: ~3.05mm per step. Approximately 60% of points use the 9-bit short path.
